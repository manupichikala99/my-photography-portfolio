# Image Upload Guide - peakthrumylens Portfolio

## Dynamic Loading (No Code Changes!)

**Gallery.js, Featured.js, Travel.js auto-load ALL images** from folders.

## Folders (paste directly):
```
src/images/portraits/     ← Portraits
src/images/nature/        ← Nature  
src/images/travel/        ← Travel
src/images/               ← Profile pic (Hero/Navbar/About)
```

## Quick Steps:
1. **Drag JPEG/PNG/SVG** to folder
2. **Save** open files (Ctrl+S)
3. **Refresh** browser (localhost:3000)

**Auto appears in:**
- ✅ Gallery (all categories)
- ✅ Featured (random mix)
- ✅ Travel component

## Tips:
- Max file size: ~5MB recommended
- Naming: Optional (peakthrumylens-001.jpg)
- Format: .jpg .jpeg .png .svg
- Backup: Git commit before bulk upload

Live preview: http://localhost:3000/#gallery
